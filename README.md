# 📊 Task Automation Tracker (Google Sheets)

## 📌 Overview

This project is a task automation system built using Google Sheets and Google Forms. It automates task tracking, prioritization, and reporting using formulas and dashboards.

---

## 🚀 Features

* 📥 Automated data entry using Google Forms
* ⚙️ Dynamic priority assignment using formulas
* 🎯 Conditional formatting:

  * 🔴 Overdue tasks
  * 🟡 Due soon tasks
  * 🟢 Completed tasks
* 📊 Dashboard using Pivot Tables and Charts (including aggregate metrics like total tasks)
* 📈 KPI metrics (Total, Completed, Pending, Completion %)
* 🔄 Fully automated using ARRAYFORMULA

---

## 🧠 Tech Used

* Google Sheets
* Google Forms
* Functions:

  * IF, AND, TODAY
  * COUNTIF, COUNTA
  * ARRAYFORMULA
* Pivot Tables
* Conditional Formatting

---

## 🔄 Workflow

Google Form → Form Responses → Tracker Sheet → Dashboard
